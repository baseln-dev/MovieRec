# MovieRec

MovieRec is a SvelteKit web app that connects to The Movie Database (TMDB) API to help users track watched movies and discover personalized recommendations.

## Tools & technologies

**Frontend**
- SvelteKit + Svelte
- Vite
- Tailwind CSS (PostCSS pipeline)

**Backend / Server**
- SvelteKit server routes
- Node.js
- PostgreSQL

**Auth & security**
- Email + password authentication
- Email verification and 2FA (TOTP)
- Session cookies

**External services**
- TMDB API for movie data
- Resend for transactional email

## How the website was built

1. **Project setup**
	- Bootstrapped with SvelteKit and Vite.
	- Tailwind CSS configured through PostCSS for styling.

2. **Authentication flow**
	- Users sign up with email and password.
	- Email verification is required before accessing protected features.
	- Optional 2FA setup and verification using TOTP.

3. **Data storage**
	- User accounts, sessions, and watchlists stored in PostgreSQL.
	- Server-side queries are handled via `pg`.

4. **Movie discovery**
	- TMDB is used for trending, popular, and search results.
	- Recommendations are generated by combining watched history and TMDB suggestions.

5. **UI & UX**
	- Shared layout with navigation and search.
	- Dedicated pages for Discover, Recommendations, and Watched movies.

## Environment variables

Created a `.env` file at the project root and provided:

```
RESEND_API_KEY=
ENCRYPTION_KEY=
```
